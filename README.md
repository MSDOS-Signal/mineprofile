# myvue

## 你可以下载下来，切换到这个根目录下，终端运行来启动这个项目

```
npm run serve
```

## 进入之后

### 默认用户名
```
root
```

### 默认密码
```
Key-122
```

## 如果不行，下面是官方的命令，依次执行就行（在根目录下）
```
yarn install
```

### Compiles and minifies for production
```
yarn build
```

### Compiles and hot-reloads for development
```
yarn serve
```
