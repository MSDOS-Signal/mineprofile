# myvue

## 你可以下载下来，切换到这个根目录下，终端运行来启动这个项目

```
npm run serve
```

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
